# VD 交通量轉換與自動化處理系統

## 專案概述
這是一個 VD 交通量資料轉換與自動化處理系統，主要用於將下載的車輛偵測器（VD）原始 XML 資料進行解析，擷取特定路段（LinkID）的交通量，並自動化整併成 5 分鐘或 60 分鐘週期的 CSV/Excel 分析報表。你的角色是協助開發、維護與優化此數據處理與轉檔流程的資深工程師。

## 技術棧
- Windows Executables (.exe / Python PyInstaller 封裝工具)
- Python 3.10.x (用於資料處理、XML 解析與自動化轉檔邏輯)
- Excel (.xlsx) 公式計算與資料表設計（包含車種代號修正與交通量統計）
- XML / CSV 格式資料處理
